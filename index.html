How to use Terminal ?

Memo des commandes LINUX

  • cd: Deplacement dans l'arborescence des fichiers.
  • cd ..: Positionnement sue le repertoire precedent.
  • pwd: Nom du repertoire courant.
  • mkdir nom_du_dossier: Creation du repertoire prog.
  • cp prog1.c prog2.c: Copie du fichier prog1.c dans prog2.C
  • rmdir prog: Effacement de repertoire prog
  • cat prog.c: Liste du contenu du fichier prog.c
